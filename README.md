## Getting Started
### Prerequisites
> Make sure NodeJS is already installed with NPM

#### Clone the repository
```bash
git clone https://github.com/HugoSohm/spotifytop.git
```

#### Install dependencies
```sh
npm install
```

#### Run project
```sh
# Production
npm run start

# Development
npm run dev
```
